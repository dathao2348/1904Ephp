Sự khác biệt của include , include_one , require , require_once

    Sự giống nhau: 4 hàm này đều dùng để nạp 1 file vapf 1 file khác
    Sự khác nhau:
    - include ( khi mà quá trính nạp file bị lỗi thì chương trình vẫn chạy tiếp)
    - include_once ( giống với include nhưng chỉ nạp file 1 lần ngay cả khi gọi nhiều lần)
    - require ( khi mà quá trình nạp file bị lỗi thì chương trình sẽ dừng chạy)
    = require_once ( giống require nhưng chỉ nạp file 1 lần ngay cả khi gọi nhiều lần)
